h2. Features

h3. Scripts

Useful tools in automating tasks.

*Script syntax*
* Type any characters for filling up forms
* Special keyboard operations are enclosed in curly braces. Example: {TAB}, {ENTER}
* Using pipe "|" will insert a second pause in operation

h3. Workspaces

Set of tools in moving around folders in commandline by using only a key word.

h3. Clipboard

Provides basic copy and pasting files in commandline
